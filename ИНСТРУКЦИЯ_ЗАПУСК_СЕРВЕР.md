# Пошаговая инструкция: запуск бота на сервере Ubuntu 24/7

Репозиторий: https://github.com/chapkinalex/davaj_igrat_bot

Выполняйте команды по порядку, подключаясь к серверу по SSH.

---

## Шаг 1. Подключиться к серверу по SSH

На своём компьютере откройте терминал (PowerShell, cmd или приложение для SSH) и выполните:

```bash
ssh ubuntu@АДРЕС_ВАШЕГО_СЕРВЕРА
```

Подставьте свой логин (например `ubuntu`) и адрес сервера (IP или домен). Введите пароль, когда попросит.

---

## Шаг 2. Установить Git (если ещё не установлен)

```bash
sudo apt-get update
sudo apt-get install -y git
```

---

## Шаг 3. Склонировать репозиторий

Перейдите в домашнюю папку и клонируйте проект (папка будет называться `davaj_igrat_bot`):

```bash
cd /home/ubuntu
git clone https://github.com/chapkinalex/davaj_igrat_bot.git davaj_igrat_bot
cd davaj_igrat_bot
```

Если репозиторий приватный, введите логин и пароль (или токен) GitHub по запросу.

---

## Шаг 4. Создать файл .env с токеном бота

Скопируйте шаблон и откройте файл для редактирования:

```bash
cp .env.example .env
nano .env
```

В файле замените `your_bot_token_here` на **ваш реальный токен** от @BotFather. Должна получиться одна строка вида:

```
BOT_TOKEN=8519012332:AAH...
```

Сохраните: **Ctrl+O**, Enter, затем выйдите: **Ctrl+X**.

---

## Шаг 5. Запустить скрипт развёртывания

Скрипт установит зависимости, создаст venv, настроит systemd и запустит бота. Выполните (подставьте своё имя пользователя, если не `ubuntu`):

```bash
chmod +x deploy_on_server.sh
./deploy_on_server.sh ubuntu
```

Когда попросит пароль для `sudo` — введите пароль пользователя сервера. Дождитесь сообщения «Готово».

---

## Шаг 6. Проверить, что бот работает

```bash
sudo systemctl status davaj_igrat_bot.service
```

В выводе должна быть строка **active (running)** зелёным. Значит, бот запущен и работает 24/7.

---

## Полезные команды потом

- **Посмотреть логи бота:**
  ```bash
  tail -f /home/ubuntu/davaj_igrat_bot/logs/bot_stderr.log
  ```

- **Перезапустить бота** (после смены кода или .env):
  ```bash
  sudo systemctl restart davaj_igrat_bot.service
  ```

- **Остановить бота:**
  ```bash
  sudo systemctl stop davaj_igrat_bot.service
  ```

- **Обновить код из GitHub и перезапустить:**
  ```bash
  cd /home/ubuntu/davaj_igrat_bot
  git pull
  sudo systemctl restart davaj_igrat_bot.service
  ```

---

## Если бот не запустился

1. Проверьте, что в `.env` указан правильный токен:
   ```bash
  cat /home/ubuntu/davaj_igrat_bot/.env
  ```

2. Посмотрите ошибки в логе:
   ```bash
  tail -50 /home/ubuntu/davaj_igrat_bot/logs/bot_stderr.log
  ```

3. После исправления `.env` перезапустите сервис:
   ```bash
  sudo systemctl restart davaj_igrat_bot.service
  ```

Если папка у вас не `/home/ubuntu/davaj_igrat_bot`, а другая (например вы клонировали в `davai_sigrai_bot`), замените путь в командах на свой. Скрипт `deploy_on_server.sh` сам подставит текущую папку в unit systemd.
